# SPEC — 랜딩 Phase 0 (`/soccer`) · PC + 모바일 통합

- **PC**: ≥1200px 기준. 콘텐츠 컨테이너 `max-width:1200px; margin:0 auto; padding:0 32px`.
- **모바일**: 390px 기준(≤480px). 좌우 패딩 22px. 반응형 브레이크포인트 1개(약 768px)로 두 레이아웃 전환 권장.
- **카피(따옴표 안 텍스트)는 PC·모바일 동일. 임의 수정 금지.**

## 섹션 순서 (공통, 임의 변경 금지)

## 공통 규칙
- 모든 pill/버튼 `white-space:nowrap`. 한글 문단은 `word-break:keep-all`(모바일 필수).
- 앵커 스크롤 smooth, 섹션 `scroll-margin-top` PC 80px / 모바일 70px.
- 오렌지 `#FF6B35`는 CTA류에만 (hover `#e85a26`).
- 히어로 사진: CMS/정적 자산으로 교체 가능하게. Pexels 기본 이미지는 임시.
- 터치 타겟 최소 44px (모바일).

## 완료 기준 체크리스트
- [ ] 두 뷰포트(≥1200px / ≤480px)에서 섹션 순서·카피 동일, 레이아웃 명세 일치
- [ ] 헤더 투명→solid 전환 (PC 40px / 모바일 30px 임계값), 모바일 메뉴 열림 시 solid 유지
- [ ] 히어로: 사진 로드 실패 시에도 네이비 배경으로 텍스트 가독성 유지 (배경색 `#1c2b4a` 폴백)
- [ ] 오버레이 아래 사진 grayscale(50%), 텍스트 대비 WCAG AA 이상
- [ ] 버튼 줄바꿈 없음, 390~480px / 1200~1440px에서 깨짐 없음

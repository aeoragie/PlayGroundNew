## 결정표 (핵심 — 역할 혼용 금지)
- **탭** = 동급 뷰 전환 (2~6개, **URL 동기화 필수** — 딥링크·뒤로가기): 팀 공개홈 6탭 · 선수 프로필 4뷰
- **세그먼트** = 같은 뷰의 데이터 축 전환 (SearchFilter 참조): 대회/리그 · 연령대
- **아코디언** = 한 페이지 안 긴 목록 접기 (뷰 전환 아님): Records 지난 회차 · FAQ
- 중첩 상한: **탭 1단 + 세그먼트 1단** (팀 공개홈 기록 탭 + 시즌 세그먼트가 상한) — 탭 안에 탭 금지

## 페이지 탭 (언더라인형)
- 선택 = 800 + 네이비 `#23408e` 2.5px 언더라인 / 비선택 = 700 `#8a93a6` · 바 하단 1.5px `#e6e8ee`
- URL: `/team/{slug}/players` 식 경로 세그먼트 · 탭 전환 시 스크롤 최상단 복귀
- 카운트 뱃지는 승인 대기류만
- **모바일**: 높이 44px, 가로 스크롤(스크롤바 숨김, 바 끝 잘림이 힌트), 선택 탭 화면 밖이면 자동 스크롤 인. **스와이프 탭 전환 금지**(가로 스크롤 콘텐츠와 충돌)

## 아코디언
- **다중 열림 허용** (단일 강제 금지 — 비교하며 봄) · 기본 = 최신 1개만 열림
- 헤더: 행 전체 탭(PC 패딩 13px / 모바일 min-height 56px, 메타는 제목 아래 줄) + 셰브론 180° 회전(.2s)
- 열림 상태는 URL에 넣지 않음(임시 상태) · 열림 시 스크롤 유지(탭과 반대)

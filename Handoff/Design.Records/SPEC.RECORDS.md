# SPEC — 경기기록 Records `/records` (PC + 모바일)

공통: 배경 `#f4f6fa` · 카드 흰+1.5px `#e6e8ee`+radius 13~16 · 컨테이너 PC max-width 1100px/패딩 32px, 모바일 480px/16px · 한글 nowrap/keep-all · 이모지 금지(모노톤 SVG stroke 1.8, 유튜브 아이콘만 공식 레드) · GNB 텍스트 워드마크(PlayGround 네이비 + Soccer 연회색).
**카피는 레퍼런스 원문 그대로. README의 설계 결정 5개 우선.**

## 목록
- 타이틀 "2026 시즌"(현재 연도 자동) + [대회|리그] 세그먼트(PC 인라인 / 모바일 풀폭) + 검색 인풋 + "지난 시즌 (기록)" 링크
- 상태 범례 3색 + "내 연령(U15)만 (보기)" 토글: on=teal 톤 배경
- **연령 아코디언**: 헤더(연령 + "내 연령" 뱃지 + 설명 + 개수 뱃지 + 셰브론 회전). 내 연령 = 최상단 + 자동 펼침 + `rgba(46,196,182,.45)` 테두리
- **대회 행**: 상태 컬러 바(4px, radius 99) + 전국(오렌지톤)/지방(네이비톤) 뱃지 + 대회명 + 형식·팀수 + 상태 뱃지(진행중=teal 채움/예정·종료=회색). 종료 행 opacity .55. PC 1줄 / 모바일 2줄(이름 위, 뱃지·메타 아래)
- 리그 세그먼트: 아코디언 내부에 지역 그룹 헤더(`#f8f9fb` 배경, 11px 800)
- 정렬: 상태(진행중 0/예정 1/종료 2) 오름차순

## 지난 시즌 아카이브
- 뒤로("2026 시즌으로") + 타이틀 + 연도 칩(2025~2021, 선택=네이비 채움)
- 대회 행: 회색 상태 바 + 범위 뱃지 + 대회명 + 연령·형식·팀수 + **"우승" 뱃지(오렌지톤) + 우승팀명**. 클릭 → 상세(대회 정보 탭부터)

## 상세 히어로
네이비 그라디언트(160deg) + 장식 원 + 뱃지(진행 상태=teal 채움, 나머지 흰 12%) + 대회명(PC 25px/모바일 18px 800) + 메타 1줄

## 대회 정보 탭
- 통계 바(네이비 `#1c2b4a`): "총 경기 · 종료" + "득점 · 경기당" + 규정 PDF 버튼(파일 아이콘)
- 개요 카드: 주최/기간/참가/방식/경기시간/(구장/순위 결정)/출처 — key-value 행
- 수상 카드: 종료 시 우승(오렌지톤)/준우승(회색)/페어플레이(teal톤) + 팀 링크. 미종료 시 "수상 내역은 대회 종료 후 제공됩니다"
- 역대 우승팀: 연도 + 팀 링크

## 조별 스테이지 (예선/1차 풀리그/2차 스플릿 공용)
- 조 선택 칩 그리드(PC minmax 64px / 모바일 가로 스크롤), 선택=네이비 채움
- 2차 스플릿 탭: 상단 안내 "2차 스플릿리그는 1차 풀리그 결과(조 1~4위)로 재배치됩니다…"
- 조 순위표: PC 8열(순위/팀/경기/승점/승/무/패/득실) / 모바일 5열(승·무·패 통합, 득실 제거). 진출권 행 = teal 랭크 뱃지 + `rgba(46,196,182,.05)` 배경 + 하단 범례
- 경기: 라운드 필터(전체/1R/2R/3R) + 카드(일시·구장·상태 / 홈-스코어-원정, 승팀 800 강조). PC 1줄 / 모바일 2줄

## 토너먼트 탭
- 라운드 칩(PO/16강/8강/4강/결승) + 안내 "득실·조 순위는 예선 기준 · PK는 괄호로 표기"
- 매치 카드(PC 2열 grid minmax 250px / 모바일 세로): 상태+일시·구장 / 팀명(승자 800)+스코어 칩(승자=네이비 채움, PK="1 (4)") / 하단 조 순위 메타

## 리그 경기 탭 (리그형)
- 순위표(왕중왕전 진출 상위 3팀 teal) + 월별 필터(전체/4월/5월/…) + 경기 카드 (조별과 동일 문법)

## 미디어 탭
- 서브탭 pill "영상 N" / "인터넷 뉴스 N" — **개수 표기 필수**
- 영상 카드: 16:9 VS 배너(네이비 그라디언트 + 팀명 VS 팀명 + 종류 뱃지 + 재생 원) + 유튜브 아이콘 + 제목 + 메타
- 뉴스 빈 상태: 점선 박스 "등록된 뉴스가 없습니다" + 부연

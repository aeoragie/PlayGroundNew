# Handoff: 경기기록 Records (공개 데이터 서비스)

> 대상 저장소: `C:\Workspace\PlayGroundNew` · 압축 해제 위치: `Handoff\Design.Records`
> 이 번들의 HTML은 **디자인 레퍼런스(인터랙티브 프로토타입)**입니다. 코드베이스 환경(Blazor WASM + Tailwind)에서 재구현하세요. 토큰은 기존 핸드오프와 동일.

## Overview
개발 로드맵 5단계. 라우트 `/records` — **비로그인 전면 무료 공개**, 대중 유입 입구이자 가입 전환 플라이휠의 시작점.
레퍼런스(nextplayer.co.kr) 분석 후 사용자 승인된 5개 설계 결정이 반영됨 (아래 "설계 결정" 필수 숙지).

## 설계 결정 (사용자 승인 — 임의 변경 금지)
1. **올해 시즌만 기본 노출** — 연도 필터 없음. "지난 시즌 기록 →"으로 아카이브 화면 분리(연도 칩 + 종료 대회 목록 + 우승팀)
2. **상태 필터 제거, 색으로 표시** — 행 좌측 컬러 바(진행중 teal `#2EC4B6` / 예정 네이비 `#23408e` / 종료 회색 `#d5d9e2` + 행 opacity .55) + 범례. 정렬은 진행중→예정→종료 자동
3. **로그인 개인화 = 우선 배치** — 내 연령(자녀) 아코디언 최상단 고정 + 자동 펼침 + teal 테두리 + "내 연령" 뱃지. 숨김은 "내 연령만 보기" 토글(opt-in)만. 타 연령을 강제로 숨기지 않음
4. **[대회 | 리그] 세그먼트 분리** — 대회=단기 컵(상태 중심), 리그=장기(연령 내 지역 그룹 헤더: 서울/인천…)
5. **상세 탭 = 대회 형식별 가변** (아래 표)

| 형식 | 탭 구성 | 비고 |
|---|---|---|
| 조별+토너먼트 (cup) | 대회 정보 / 예선 / 토너먼트 / 미디어 | 예선=조 선택 칩(1~14조) + 조별 순위·경기 |
| 풀리그+스플릿 (split) | 대회 정보 / 1차 풀리그 / 2차 스플릿 / 미디어 | 2차 탭에 재배치 안내 문구 필수 |
| 리그 (league) | 리그 경기 / 미디어 | 순위표 + 월별 필터 |

## 화면 구조
- **목록**: 2026 시즌 타이틀 + [대회|리그] 세그먼트 + 검색 + 상태 범례 + 내연령 토글 → 연령 아코디언(U15 최상단) → 대회 행(상태 바 + 전국/지방 뱃지 + 형식·팀수 + 상태 뱃지)
- **아카이브**: 연도 칩(2025~) + 종료 대회 리스트(우승팀 컬럼) → 클릭 시 상세
- **상세 공통**: 네이비 그라디언트 히어로(상태·범위·연령 뱃지 + 대회명 + 메타) + 가변 탭
- **대회 정보 탭**: 통계 바(총 경기·종료 / 득점·경기당) + 규정 PDF 링크 + 개요(주최·기간·방식·경기시간·출처) + 수상(종료 시만, 미종료 시 "대회 종료 후 제공") + 역대 우승팀
- **조별 스테이지**: 조 선택 칩 → 조 순위표(진출권 teal 행) + 라운드 필터 + 경기 카드
- **토너먼트**: 라운드 칩(PO/16강/8강/4강/결승) + 매치 카드(승자 네이비 스코어, **PK는 괄호 표기** "0 (3)", 조 순위 메타)
- **미디어**: 영상/뉴스 서브탭 + **개수 뱃지**(빈 탭 클릭 방지) + VS 배너 카드(유튜브 아이콘) + 뉴스 빈 상태 디자인
- **모바일 차이**: 세그먼트 풀폭, 탭·칩 가로 스크롤, 순위표 득실 열 제거(승·무·패 한 열), 경기 카드 2줄 구조

## 전환 훅 (플라이휠)
- 헤더: 로그인 시 "김OO 님 · U15 보호자" / 비로그인 시 "로그인 없이 누구나 볼 수 있어요" + 로그인 버튼
- 팀명 클릭 → 공개 팀 홈페이지, 선수명 클릭 → 공개 선수 프로필
- 푸터: 무료 공개 원칙 + 출처 표기(KFA 공식 · 대회 주최자 입력 · 팀 입력)

## 백엔드 연결점
- Tournament.Format(cup/split/league)이 상세 탭 구성을 결정
- DataSource(KfaApi/AgentHosted/Manual) → 출처 뱃지. 별도 대회 서비스 데이터는 공유 DB로 유입
- 개인화: 로그인 계정의 자녀 연령(FamilyLink → Player.AgeGroup) 기준
- 미디어 영상 = 팀/주최자 등록 유튜브 링크 (MatchId 연결)

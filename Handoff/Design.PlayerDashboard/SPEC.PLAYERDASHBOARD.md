공통: 배경 `#f4f6fa` · 카드 흰 배경+1.5px `#e6e8ee`+radius 14~16 · 섹션 타이틀 PC 23px/모바일 19px 800 · 주 버튼(오렌지 #FF6B35) 화면당 1개 · 필터 pill 선택=네이비 채움 · 모노톤 라인 SVG(stroke 1.8) · 이모지 금지 · 한글 nowrap/keep-all.
**카피는 레퍼런스 원문 그대로.**

## 1. 프로필
- 헤더: "프로필" + 오렌지 "프로필 수정"
- PC: [본문 카드 | 우측 340px] 2컬럼 / 모바일: 세로 스택(기본 카드 → 공개 프로필 → 공개 설정 → 가족 계정)
- **기본 카드**: 3:4 프로필 사진(PC 120px/모바일 92px, image slot) + 이름 + **"가족 연결됨" teal 뱃지** + 정보 그리드(포지션·등번호 / 생년·학년 / 소속)
- **항목별 공개 설정**: 행(항목명+값+토글 pill). 토글: 공개=teal 톤 / 비공개=회색. 캡션 "보호자만 변경할 수 있어요". 항목: 키/몸무게/주발/학교/보호자 연락처(마스킹 010-****-1234). 기본값: 키·몸무게·주발 공개, 학교·연락처 비공개
- **공개 프로필 카드**(네이비 그라디언트): URL + "이번 주 방문 34회 · 에이전트 열람 요청 1건" + [공개 뷰 보기(흰)/선수 카드 공유(고스트)]
- **가족 계정 카드**: 보호자(관리자 teal 뱃지) / 본인(열람 회색 뱃지, "만 14세 이상 시 관리 권한 이전 가능") + ＋ 가족 초대(점선)

## 2. 커리어
- 헤더 + 오렌지 "＋ 이력 추가"
- 타임라인: 도트(현재=teal/과거=회색) + 세로 라인 + [팀명 + "현재 소속" teal 뱃지 + 특이 뱃지(예: "서울 지역 대표 선발" 오렌지톤) / 기간·역할 / 노트]
- **검증 뱃지**: "팀 확인됨"(teal) / "본인 입력"(회색) — PC 우측, 모바일 본문 하단
- 캡션: 팀 관리자 확인 → 팀 확인됨 → 공개 프로필 신뢰 표시

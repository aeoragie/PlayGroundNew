## 요소별 결정표 (무엇을 쓸까)

**선택 컨트롤**: 즉시 켬끔 → 스위치(10) / 폼 동의·다중 → 체크박스(4) / 폼 상호배타 ≤4 → 라디오 카드(4) / 5개 이상 값 → 셀렉트(4, 모바일=바텀시트) / 데이터 축 전환 → 세그먼트(5) / 조건 좁히기 → 필터 칩(5)

**날짜**: 생년월일 → 자동 포맷 입력(4·11) / 일정 → 캘린더(11) / 연도 → 칩(11)

**목록**: 열 5개 이상 → 테이블(6) / 행 탭 이동 → 내비게이션형 행(6) / 행 안 버튼 → 액션형 행(6) / 결과 표시 → 정보형 행(6) / 페이징 → 더 보기(6), 관리자만 숫자(14)

**화면 구획**: 동급 뷰 전환 → 탭(7, URL 동기화) / 긴 목록 접기 → 아코디언(7) / 다단계 진행 → 스텝퍼(15)

**메시지**: 완료 통지 → 토스트(3) / 되돌릴 수 없는 확인 → 모달(3) / 폼 오류 → 인라인(4) / 운영 공지 → 배너(15) / 개념 설명 → 인포 팝오버(13) / 아이콘 레이블 → 호버 툴팁(13, PC만) / 학습 유도 → 첫 사용 힌트(13)

**상태**: 로딩 → 스켈레톤(1, 스피너는 버튼 제출만) / 0건 → 빈 상태(2) / 신원 표시 → 아바타(9) / 상태 표시 → 뱃지(9, 행당 1개)

**액션 모음**: 계정 → 계정 메뉴(8) / 행·카드 부가 액션 → ⋯(8) / 이미지 → 업로더 3형(12)

## 전역 불변 규칙 (전 패턴 공통)
1. 오렌지 `#FF6B35` = 랜딩 CTA + 카운트 뱃지 + 대기·주의 톤만 — 그 외 주 버튼은 네이비 `#23408e`
2. teal `#2EC4B6` = 긍정·완료·켬 / violet `#6d4bd8` = 에이전트 전용
3. 한글: 버튼·메타 nowrap / 문단 keep-all · 이모지 금지(모노톤 라인 SVG stroke 1.8)
4. 모바일: 터치 44px 이상·입력 16px·앵커 팝업 대신 바텀시트·스와이프 액션 금지
5. URL 동기화: 탭·필터·페이지 = URL / 아코디언·메뉴 열림 = URL 제외
6. 상태 연쇄: 로딩(스켈레톤) → 0건(빈 상태) → 실패(인라인/토스트) 순으로 항상 처리

# SPEC — 공개 팀 홈페이지 `/team/{slug}` (PC + 모바일)

공통 토큰은 기존 핸드오프와 동일. 컨테이너 PC max-width 1120px/패딩 32px, 모바일 480px/16px.
카드 = 흰 배경 + 1.5px `#e6e8ee` + radius 14~16. 탭 선택 = `#23408e` 보더/글자. 한글 nowrap/keep-all 규칙 필수.
**카피는 레퍼런스 HTML 원문 그대로.**

## 히어로
| | PC | 모바일 |
|---|---|---|
| 커버 높이 | 260px | 150px |
| 엠블럼 박스 | 118px, radius 26, 겹침 -72px | 84px, radius 19, 겹침 -44px |
| 팀명 | 30px/800 | 20px/800 |
| CTA | 우측 "입단 문의"(오렌지 46px) + 공유 버튼 | 하단 고정 바(공유 48px + 오렌지 풀폭) |

오버레이: `linear-gradient(180deg, rgba(28,43,74,.2~.25), rgba(28,43,74,.7~.78))`. 텍스트 블록 `position:relative;z-index:5` (엠블럼이 커버 뒤로 깔리는 버그 방지).

## ① 소개
- PC: [팀 소개+요약 칩 | 공식 채널 340px] 2컬럼 → 핵심가치 → 코칭스태프. 모바일: 전부 세로 스택(소개→핵심가치→코칭스태프→공식 채널).
- 팀 소개: 문단 + 칩(월 회비 / 훈련 요일 / 구장 / 선수단 수)
- 핵심가치 3장: PC `border-top:3px solid #2EC4B6` 카드 grid / 모바일 `border-left` 행. 부제 "학부모님과 선수에게 드리는 약속입니다"
- 코칭스태프: 이니셜 아바타 + 이름·직책 + 경력 + KFA 자격 뱃지 + **지도 철학 인용문**(teal 좌보더) + 실적 칩(PC만) + SNS 버튼(인스타 그라디언트·유튜브 레드 공식 아이콘)
- 공식 채널: 유튜브·인스타 행(공식 아이콘 24~26px + 채널명 + 설명 + 열기 링크)

## ② 선수단
- 연령 pill(U12/U15/U18) + 사진 카드 grid (PC minmax 200px / 모바일 2열)
- 카드: 4:3 사진(등번호 원 오버레이) + 이름 + 포지션·학년. **Claimed 선수만 "공개 프로필 →"**. Claim 뱃지 비노출.
- 하단 캡션: "공개 프로필이 있는 선수만 상세 기록으로 연결됩니다…"

## ③ 시즌성적
- 요약: PC 4카드(전적 네이비 그라디언트/득점/실점/순위▲) / 모바일 2카드(전적 / 득실·순위)
- 최근 경기 리스트: 대회 pill + 날짜·장소 + 팀명 score 상대 + 승무패 뱃지(승 teal 채움/무 회색/패 회색 채움). "경기기록에서 전체 보기 →" → Records
- 경기 영상: 16:9 썸네일 카드(재생 원·길이 뱃지·유형 pill·유튜브 아이콘)

## ④ 모집
- 모집중 공고 카드(제목+모집중 뱃지+설명+조건 칩+**지원하기 네이비 버튼**) / 마감 공고(회색, opacity .75)
- "모집 공고가 없어도 문의할 수 있어요" 카드 (PC: 입단 문의 남기기 아웃라인 버튼)

## ⑤ 진학·진로
- 요약 3카드: 프로 산하 이적 / 축구부 진학 / 상급팀 승격 (모바일 3열 압축)
- 연도별 타임라인 행: 연도(네이비) + 제목 + 상세 + 유형 태그(프로 산하=오렌지톤/축구부=네이비톤/승격=teal톤)
- 캡션: "선수 개인이 공개에 동의한 사례만 표시됩니다."

## ⑥ 리뷰
- 헤더: 평균 4.8(30px) + 오렌지 별 + "리뷰 12개". 부제 "재원 중이거나 재원했던 가족만 작성할 수 있어요"
- 리뷰 카드: 이니셜 아바타 + "이○○ 학부모" + 메타(연령·재원기간) + **"재원 확인됨" teal 뱃지** + 별점 + 본문. 4★ 리뷰 포함(진정성)
- 캡션: "리뷰는 재원 이력이 확인된 계정만 작성 가능하며, 팀은 삭제할 수 없고 답글만 달 수 있습니다."

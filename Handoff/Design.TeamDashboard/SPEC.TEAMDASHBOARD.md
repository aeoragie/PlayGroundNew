# SPEC — 팀 대시보드 P0 (PC + 모바일)

본문 1~6장은 PC(≥1024px) 기준. 모바일(≤480px) 차이는 맨 아래 "모바일 적응" 장 참조 — 카피·색·뱃지 규칙은 동일.

공통: 배경 `#f4f6fa` · 카드 = 흰 배경 + 1.5px `#e6e8ee` + radius 14~16px · 섹션 타이틀 23px/800 · 주 버튼(오렌지) 화면당 1개, 높이 42px · 필터 칩 pill(선택 = 네이비 `#23408e` 채움, 미선택 = 흰+보더) · 테이블 헤더 12px/800 `#8a93a6`, 배경 `#f8f9fb`.
아이콘: 모노톤 라인 SVG(stroke 1.8). 이모지 금지. 한글 메타 `nowrap`/`keep-all`.
**카피는 레퍼런스 HTML 원문 그대로.**

## 레이아웃
- GNB 60px 네이비 `#1c2b4a` sticky: 로고+워드마크, "팀 관리자 모드" pill, 우측 [외부링크 아이콘+공개 홈페이지 보기 / 벨 / 아바타(이니셜, teal 원)]
- 사이드바 230px `#fdfdfc` + 우측 보더: 팀 요약(실드 아이콘 44px 네이비 박스 + 팀명 + "✓ 인증팀 · 클럽 · 서울") → 메뉴(선택 = 네이비 채움+흰 글자, radius 10) → "P1 예정" 레이블 + 비활성 3항목(opacity .55)
- 콘텐츠 max-width 1100px, padding 30×34

## 1. 팀 정보
1. **기본 카드 + 공개 홈페이지 카드** (grid auto-fit minmax(340px)):
   - 기본: 엠블럼(72px 네이비 실드) + 팀명 20px/800 + 뱃지 4개(인증팀 오렌지톤 / 클럽·지역·창단 회색) + 요약 2칸(월 회비 "250,000원 · 공개" / 훈련 "주 4회 · 화목금토")
   - 홈페이지 카드(네이비 그라디언트): URL + "이번 주 방문 128회 · 핵심가치와 코칭스태프가 홈페이지 소개 탭에 그대로 노출"
2. **핵심가치 카드** (full width) — 학부모 대상 최중요 요소:
   - 헤더: "핵심가치" + teal 뱃지 "공개 홈페이지 노출중" + 캡션 "학부모가 팀을 고르는 첫 번째 기준이에요"
   - 가치 카드 3장 (grid minmax(260px), `#f8f9fb` 배경, **상단 3px teal 보더**): 제목 15px/800 + 구체적 약속 설명 13px/1.65
   - 예시: 성장 중심 지도(분기 1:1 성장 리뷰) / 출전 기회 보장(전원 50% 이상 출전 원칙·기록 공개) / 학업 병행 존중(시험 기간 조정)
3. **코칭스태프 카드** (full width, teal 뱃지 + "＋ 코치 초대"):
   - 프로필 카드 2장 (grid minmax(320px)): 이니셜 아바타(50px 네이비 원) + 이름·직책 + 경력 한 줄("지도 12년차 · 前 K리그 유스 코치") + KFA 자격 뱃지
   - **지도 철학 인용문** (좌측 3px teal 보더, 13.5px): 코치 가치관 노출 — 학부모 선택 기준
   - 실적 칩("프로 산하 이적 3명" 등) + **SNS 링크 버튼**(인스타 그라디언트 글리프·유튜브 레드 아이콘 15px + 라벨, 미등록 시 "유튜브 미등록" 회색 텍스트)
4. **공식 채널 카드** (full width, teal 뱃지 + "＋ 채널 추가"): 유튜브 행(공식 아이콘 30px + 채널명 + URL·설명 + "채널 열기 →") / 인스타 행(그라디언트 글리프 30px + @핸들 + 설명 + "프로필 열기 →")

## 2. 선수단
- 헤더: "선수단 n명" + [리스트|카드 세그먼트 토글] + "초대코드 일괄 발송" + "＋ 선수 추가"(오렌지)
- 연령 탭: U12(n)/U15(n)/U18(n) pill
- **리스트 뷰(기본)**: 테이블 6열(번호 원 32px/이름 링크/포지션/학년/연결 상태/액션). Claim 뱃지: Claimed(teal)/Pending(오렌지톤)/Unclaimed(회색). 액션: 프로필 보기/승인 대기/초대코드 보내기
- **카드 뷰**: grid minmax(210px). 상단 4:3 사진(등번호 뱃지 좌상단 오버레이) + 이름 + "포지션 · 학년"(nowrap) + Claim 뱃지 + 액션. 사진 없으면 "선수 사진" 플레이스홀더
- 하단 안내(오렌지 점선 박스): Unclaimed → 초대코드 → 가족 관리 설명

## 3. 일정
- "2026년 7월" + "＋ 일정 추가". 이벤트 행: 요일+날짜(52px 컬럼) / 유형 pill(경기=네이비, 대회=오렌지톤, 훈련=teal) / 제목+메타 / 시간
- 하단 캡션: 공개 홈페이지 자동 노출 · Google/Apple 구독 캘린더

## 4. 경기 결과
- 요약 4카드: "2026 시즌 8승 2무 1패"(네이비) / 득점 24 / 실점 9 / 리그 순위 2위 ▲1
- 필터: 전체/리그/컵/친선 (동작)
- 경기 카드: [대회 pill + 날짜·장소 + 결과 뱃지(승=teal 채움/무=회색/패=회색 채움)] / 중앙 스코어("팀명 3 : 1 상대") / 이벤트 칩("득점 김민준 ×2" — 이모지 아님 텍스트)
- 캡션: 득점·도움 → 선수 시즌 통계 자동 반영 · Records·홈페이지 노출

## 5. 경기영상
- "경기영상 n개" + "＋ 영상 등록". 필터: 전체/하이라이트/풀경기/훈련 (동작)
- 영상 카드 grid minmax(280px): 16:9 썸네일(중앙 재생 원 44px + 우하단 길이 뱃지) / [유형 pill + 유튜브 공식 아이콘 15px + "유튜브"] / 제목(2줄, keep-all) / 메타("7/5 · 경기 결과와 연결됨")
- 안내 박스: 유튜브 링크 등록 방식 · 경기 연결 시 경기 상세·홈페이지 자동 노출

## 6. 선수 모집
- 공고 카드 2장(모집중 teal 뱃지 · 마감 회색+opacity .65): 제목 + 조건 + "지원 n · 마감 날짜"
- 지원자 리스트: 이니셜 아바타 + 이름(+에이전트 추천 Violet 뱃지 `#6d4bd8`) + 메타 + 상태 뱃지(테스트 확정=teal/검토중=오렌지톤/대기=회색) + [프로필 / 테스트 요청(네이비)]
- 안내 박스(Violet 점선): 에이전트 추천 파이프라인(대기→검토중→수락/거절)

## 모바일 적응 (≤480px · 레퍼런스: Team Dashboard Mobile.dc.html)
- **네비게이션**: 사이드바 제거 → 하단 고정 탭바 5개(팀 정보/선수단/일정/경기/모집). 아이콘 20px + 라벨 10.5px, 선택 색 네이비. `env(safe-area-inset-bottom)` 패딩, 배경 블러. 콘텐츠 하단 패딩 96px.
- **경기 탭**: 결과/영상 서브탭(세그먼트 컨트롤, 선택 = 흰 배경+그림자). 주 버튼 라벨이 서브탭 따라 변경(결과 입력/영상 등록).
- **상단 바**: 엠블럼 + 팀명 + "✓ 인증팀 · 팀 관리자 모드" + 아이콘 버튼 2개(공개 홈페이지·알림, 36px). sticky.
- **팀 정보**: 카드 전부 세로 스택. 핵심가치는 좌측 3px teal 보더 행 3개. 요약 2칸(회비/훈련) 2열 유지.
- **선수단**: 리스트 기본(행: 번호 원+이름+포지션·학년+Claim 뱃지) / 카드 토글 시 2열 그리드. 토글은 아이콘 세그먼트(리스트/그리드 아이콘).
- **일정**: 날짜 컴럼 38px, 유형 pill+시간을 제목 위 한 줄로.
- **경기 결과**: 요약 4카드 → 2카드(시즌 전적 / 득실·순위)로 압축. 팀명 "FC 한강"으로 단축.
- **터치 타겟 ≥44px**, 모든 한글 메타 nowrap/keep-all, 주 버튼 높이 40px.
- 이미지 슬롯 id는 PC와 공유(동일 디렉토리 기준) — 사진 1회 등록으로 양쪽 반영.

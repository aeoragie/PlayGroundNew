# SPEC — 대시보드 허브 `/dashboard` (PC + 모바일)

공통: 배경 `#f4f6fa` · 카드 흰+1.5px `#e6e8ee`+radius 13~18 · GNB 네이비 `#1c2b4a` · 워드마크 PlayGround(흰)+Soccer(흰 45%) · 이모지 금지(모노톤 SVG stroke 1.8) · 한글 nowrap/keep-all · 오렌지 = 팀 카드 주 버튼 1개 + 알림 카운트만.
**카피는 레퍼런스 원문 그대로.**

## 라우팅 (README 규칙 우선)
역할 1개 → 스킵 리다이렉트 / General → 역할 선택 / 그 외 → 허브.

## 섹션별
### 인사
- H1 "안녕하세요, {이름} 님" (PC 24px / 모바일 20px 800) + 요약 문장(경기 N건·알림 N건 — 동적 조립, 둘 다 0이면 "오늘도 좋은 하루 보내세요" 대체)
- PC 우측 날짜 표기

### 처리가 필요해요
- 카드 테두리 `rgba(255,107,53,.35)` — 유일하게 오렌지 테두리 허용되는 카드
- 헤더: 제목 + 카운트 뱃지(오렌지 채움) + "알림 센터 전체 보기 →"
- 항목(PC grid minmax 300px / 모바일 세로): 유형 칩 원(36/32px: 연결=#fff0e9·#c24a1c, 열람=violet 톤, 결과=#f1f3f8·#23408e) + 제목 12.5px 800 + 설명 + 셰브론. 항목 전체가 딥링크
- 최대 3건 + 초과분은 알림 센터로. 0건 시 섹션 숨김

### 내 팀
- 섹션 헤더: "내 팀" + 역할 레이블 "팀 관리자" + (PC) "＋ 팀 만들기" 점선
- 카드: 그라디언트(160deg #1c2b4a→#23408e) + 장식 원 + 실드 아이콘 박스(58/48px 흰 10%) + 팀명(18/15.5px 800) + "✓ 인증팀" + 요약 1줄 + 다음 경기 1줄(teal #2EC4B6 700)
- 버튼: "팀 대시보드"(오렌지 42px) + "공개 홈페이지"(고스트 흰 10%+보더 22%). 모바일 = 2버튼 풀폭 행

### 내 자녀
- 카드: 아바타(48/42px 네이비 원) + 이름 + Claim 뱃지 + 메타 1줄
- 스탯 3칸(`#f8f9fb` radius 10): 출전/득점/도움 — **Pending은 "–"**
- 노트 1줄(방문 수·열람 요청 / Pending은 대기 안내+요청일)
- 버튼: Claimed=[선수 대시보드(네이비 채움) / 공개 프로필(아웃라인)] · Pending=[요청 상태 보기(네이비 채움)만]
- "＋ 자녀 연결" → `/claim`

### 바로가기
- PC 3카드(경기기록/팀 탐색/계정 설정) 가로형 / 모바일 2카드(경기기록/팀 탐색) 2열 세로형
- 경기기록 부제 = 개인화: "U{자녀 연령} 진행중 대회 N개"

## 완료 기준 체크리스트

- [ ] 복수 팀/복수 자녀 카드 반복, Pending 자녀 상태 표현(스탯 –, 공개 프로필 숨김)

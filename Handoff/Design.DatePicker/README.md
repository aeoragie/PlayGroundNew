# Handoff: 날짜 선택 패턴 (Date Picker)

> 대상 저장소: `C:\Workspace\PlayGroundNew` · 압축 해제 위치: `Handoff\Design.DatePicker`
> 공용 패턴 7번째. 별도 모바일 파일 없음 — 카탈로그에 바텀시트 변형 포함(생년월일·연도 칩의 모바일 규칙은 FormPatterns·SearchFilter 참조).

## 용도별 3분법 (핵심)
1. **생년월일 = 숫자 입력 자동 포맷** (FormPatterns) — **캘린더 금지**: 과거 10~15년을 화살표로 넘기게 하지 말 것
2. **일정 날짜(가까운 미래) = 캘린더** — 팀 대시보드 일정 추가 전용
3. **시즌·연도 = 칩/리스트** (Records 아카이브) — 선택지가 연 단위면 캘린더가 아님

## 캘린더 명세
- PC = 팝오버 300px / 모바일 = 바텀시트(셀 44px, 하단 확정 버튼에 선택 날짜 표시 — 지연 적용, FeedbackPatterns 시트 규칙)
- 일요일 시작 · 요일 색: 일 `#c46a5e` / 토 `#5a7fc0` / 평일 `#8a93a6`
- 셀 상태: 오늘 = teal 링(inset 1.5px `#2EC4B6`) / 선택 = 네이비 원 채움 / 과거 = 비활성 `#c9cfda`(일정 추가 시) / 경기 있는 날 = 하단 teal 도트 4px
- 하단 퀵버튼: "오늘" · "이번 주말"
- **네이티브 `<input type="date">` 금지** — 브라우저별 편차

## 시간 선택
- 날짜와 **별도 필드** (한 팝오버에 합치지 않음)
- 15분 단위 스크롤 리스트 + 직접 입력 허용
- 기본값 = 최근 일정과 같은 시각 (팀 루틴 반복 — 매주 토 14:00 훈련)
- 반복 일정(매주 X요일)은 P1 — 캘린더에 반복 토글 넣지 말고 별도 필드로

## Files

- `support.js` — 레퍼런스 실행용 (구현 대상 아님)

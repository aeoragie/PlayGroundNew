## B. 패턴 카탈로그 — 소비 안 된 것

| 핸드오프 | 상태 |
|---|---|
| **TooltipHelp** | **코드 소비 0건** — 호버 툴팁·인포 팝오버(ⓘ)·첫 사용 힌트 전부 미구현. 카탈로그 지정 사용처(Claim "내 자녀예요"·열람 승인 30일 만료·인증팀·Records 집계 기준)가 **전부 기존 화면**이라 ⓘ 팝오버 4곳 붙이는 1회 작업 가치 있음 |
| PaginationBreadcrumb | 브레드크럼은 에이전트 심사 1곳뿐. 숫자 페이지네이션은 사용처(관리자 테이블) 자체가 없어 정상. Records 상세는 2단이라 "GNB+탭으로 충분" 부합 — **경기 상세 화면이 생기면** 3단 브레드크럼 필요 |
| TableList | 원칙(더 보기 20개·행 3형·모바일 카드 변환)은 화면별 반영. 미구현: **순위표 "내 팀 행" 마킹+스크롤 밖 상단 고정 요약** · 정렬 가능 열(동시 1열) · "전체 기록 보기" 가로 스크롤 풀버전 |
| TabAccordion | 탭 URL 동기화·세그먼트·다크 GNB 탭 소비됨. 아코디언은 Records 연령 그룹 자체 구현 — 공용 추출은 두 번째 사용처(FAQ 등)가 없어 규칙상 정상 |
